Detailed Description

private pointer-to-implementation class containing all backend methods

This class performs all behind the scenes operations acting as the backend to the user called FDP::DataPipeline class. This structure has been chosen to allow for unit tests to be created to test all methods including private ones.

Warning
The class should not be used directly during implementation, but rather via an FDP::DataPipeline instance.
